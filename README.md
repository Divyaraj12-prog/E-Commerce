# 🛒 React E-Commerce (Redux Demo)

A frontend-focused e-commerce application built to demonstrate global state management using Redux and clean UI design with React.

🔗 Live Demo: (https://e-commerce-ten-peach-93.vercel.app/)  
🔗 GitHub Repo: (https://github.com/Divyaraj12-prog/E-Commerce.git) 

---

## ✨ Features

- 🛍️ Product listing with static data
- 🛒 Add to cart / remove from cart
- 🔢 Quantity management
- 💾 Cart persistence using localStorage
- ⚡ Global state management using Redux
- 📱 Fully responsive UI

---

## 🛠️ Tech Stack

- React
- Redux Toolkit
- Tailwind CSS
- JavaScript

---

## ⚙️ Project Approach

This project is focused on frontend architecture and state management.

Instead of using a backend, static product data is used to ensure consistent behavior across all users. The primary goal is to demonstrate how Redux manages global state such as cart operations and UI updates.
